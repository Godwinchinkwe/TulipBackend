# Luxuria Booking Backend

Local Node.js + Express backend for Luxuria Grand Hotel bookings.

## Requirements
- Node.js (v16+ recommended)
- Local MongoDB running on default port (or change MONGO_URI in .env)

## Install
1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env` and edit values
